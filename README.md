# Skimpulse

A beautifully crafted **Flutter app** for staying up-to-date with the latest technology news and articles. Clean design, fast performance, and a delightful reading experience across iOS, Android, and Web.

![Flutter](https://img.shields.io/badge/Flutter-3.8-blue)
![Dart](https://img.shields.io/badge/Dart-3.8-blue)
![iOS](https://img.shields.io/badge/iOS-Compatible-green)
![Android](https://img.shields.io/badge/Android-Compatible-green)
![Web](https://img.shields.io/badge/Web-Compatible-green)

## Features

- **Latest Tech News** - Curated articles from top technology sources
- **Beautiful Reading Experience** - Clean, distraction-free article views
- **Multiple Themes** - Light and dark modes to suit your preference
- **Smart Sorting & Filtering** - Find articles that matter to you
- **Cross-Platform** - Native experience on iOS, Android, and Web
- **Fast & Responsive** - Optimized performance across all devices

## Installation

```bash
git clone https://github.com/nikhilGreenBot/skimpulse_app.git
cd skimpulse_app
flutter pub get
flutter run
```

## How It Works

1. **Browse Articles** - Scroll through the latest tech news on the home screen
2. **Read In-App** - Tap any article for a clean, in-app reading experience
3. **Customize Theme** - Switch between light and dark modes
4. **Sort & Filter** - Find specific topics or sort by date/relevance
