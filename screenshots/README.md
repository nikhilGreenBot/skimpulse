# Screenshots Directory

Place your app screenshots here organized by platform.

## Directory Structure

```
screenshots/
├── android/     # Android screenshots
├── ios/         # iOS screenshots
└── web/         # Web screenshots
```

## Recommended Screenshots

### Android
- Main screen (article list)
- Article detail view
- Settings/About screen
- Different themes (if applicable)

### iOS
- Main screen (article list)
- Article detail view
- Settings/About screen
- Different themes (if applicable)

### Web
- Main screen
- Responsive view (mobile/tablet)
- Article view

## Naming Convention

Use descriptive names:
- `android-main-screen.png`
- `ios-article-view.png`
- `web-responsive.png`

## Image Guidelines

- **Format**: PNG or JPG
- **Size**: Recommended 1080x1920 for mobile, 1920x1080 for web
- **Quality**: High resolution, clear and readable
- **Content**: Show real content, not placeholder text
